Toplumda sıkça kullanılan İngilizce kökenli kelimeler yerine kullanabileceğimiz Türkçe kelimeleri hatırlatır.

== Description ==

Türkçe Konuş, günlük hayatta bilinçsizce kullanılan İngilizce'den geçmiş kelimeler yerine kullanabileceğimiz Türkçe kelimeleri hatırlatan bir eklentidir. Bu eklentiyi sitenize ekleyerek sizde Türkçe'nin doğru kullanılmasına katkıda bulunabilirsiniz.

== Installation ==

1. turkce-konus klasörünü wp-content/plugins klasörüne yükleyiniz.
2. Eklentiler sayfasından eklentiyi etkinleştiriniz.
3. Görünüm/Bileşenler sayfasından temanıza ekleyip bir tema seçiniz.
*  Dilerseniz Tema ve Yön seçerek eklentiyi sitenize daha uyumlu hale getirebilirsiniz.
